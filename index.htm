Foundation-strength.js

Super-easy way to add password strength indicator to your form. For more inforamtion use the README and the source, Luke.

  • Simplest usage
  • Meter style
  • Caps lock detector

Go green enough with strong password like this #2!s1W1a$ Paste #2!s1W1a$ twice and password's score will be lower due to repeated characters. Use same #2!s1W1a$ with extra but unique chars and get partanoid password #2!s1W1a$b^F9z).

Styling bar possible with radius (default) or round classes: $('.strength').strength({meterClasses: 'round foobar'}); Use this setting to pass extra class you want e.g. `foobar` in this example.

Enable Caps lock and start typing in any field.
